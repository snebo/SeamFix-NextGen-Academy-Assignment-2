# Product Cart - Angular Component Communication

## Project Overview
This project is a product browsing interface built as part of the **Seamfix Assessment** (2nd assignment). It demonstrates core Angular concepts, specifically nested component communication using `@Input()` and `@Output()` decorators.

## Features
* **Dynamic Product Grid:** Displays a collection of products using a responsive CSS Grid.
* **Real-time Search:** A search bar that filters products by name as the user types.
* **Interactive Cart:** Users can select products to add/remove from a cart, with a live counter in the navbar.
* **Responsive Design:** - Desktop: 3 columns
  - Tablet: 2 columns
  - Mobile: 1 column
